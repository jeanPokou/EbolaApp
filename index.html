Welcome “Ebola SURVEYOR“ Pages.

Ebola Surveyor is an application to stop Ebola outreach in West Africa developed by jean Pokou pour le groupe Ivoirien IVOCARTE ABYSHOP. This application was built using Ionic the beautiful, open source front-end SDK for developing hybrid mobile apps with HTML5 and google Angular Js.

Installation

First Download the project or clone it using git
1. Install Node Js
Node js
2. Install Ionic
Using npm command line tool install ionic
$npm Install -g cordova ionic
Then Move to the directory of the project, build android or iOS and start, that’s all
$ cd your_repo_root/repo_name
$ ionic platform add android
$ ionic build android
$ ionic run android
If you dont have a connected android device the emulator will start

NB: you need Apache ant and Android development Tools in you environment path if you are on Windows, you can google how to use Cordova.
